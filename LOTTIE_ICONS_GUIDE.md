## üí° Pro Tips

1. **File Size**: Keep animations under 50KB for performance
2. **Duration**: Prefer animations under 2 seconds
3. **Loop**: Set `loop={false}` for one-time animations
4. **Colors**: Download animations that match your color palette, or edit colors in LottieFiles editor
5. **Style Consistency**: Use similar animation styles across all icons
6. **Fallback**: Always have Lucide icons as fallback if Lottie fails to load
